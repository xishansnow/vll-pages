🔥 单机全局路径规划方法索引帖

Created2025-01-19|Updated2025-01-17|任务规划单机规划
|Word Count:5k|Reading Time:14mins

路径规划问题通常被视为一个 “最优路径或最短路径问题”，这与将任务分配视为 “组合优化问题” 截然不同。具体来说，路径规划为某架飞行器找到一条从起点到终点的路径，使其能够避开障碍物、最大化效率或满足其他优化目标。路径规划在理想空间中寻找一条从起点到终点的可行路径、次优路径或最优路径，其结果是一个航路点序列，并且通常作为行动规划（ Motion Planning ）问题的初值。

🔔 本帖仅针对单一飞行器（机器人）的全局路径规划问题进行综述 🔔 。

一、基础流程

  • 环境建模
    • 将整个环境进行建模，包括飞行器的起始位置、任务的起始位置和目标位置、障碍物位置等。
  • 路径规划
    • 对于每个飞行器，根据分配的任务，使用合适的路径规划算法（如 A* 算法、Dijkstra算法等）规划其从起始位置到目标位置的路径。需要考虑避开障碍物、最小化路径长度或时间等优化目标。
  • 路径执行
    • 将规划好的路径转换为飞行器可以执行的指令序列，使得飞行器按照规划的路径执行任务。

二、算法的评价准则

  • 完备性
    • 算法规划出的结果是否为问题的解，所有可能的解是否都能够被算法找到。
  • 最优性
    • 保证规划结果是否最优解，因此被划分出最优解、次优解、可行解，而次优解又进一步被划分为有界次优解、无界次优解。
  • 高效性
    • 算法找到预期路径所需时间

三、主要算法

路径规划算法可以根据环境建模方法、是否考虑随机性等进行多个维度的分类，通常人们习惯于划分为基于采样的方法、基于搜索的方法、 基于个体的方法 和基于群体的方法：

3.1 基于样本的方法

方法特点

  • 初始条件
    • 起点和终点 + 环境 （ 连续空间表示或离散空间表示 ）
  • 寻路过程
    • 路径点的选择具有一定随机性
  • 输出结果
    • 一个可行解（ 可能不是最优的 ）

典型算法

  • 快速探索随机树（ RRT ）：以状态空间中的起点作为根节点，然后通过随机逐渐增加叶子节点来生成随机扩展树。如果新生成的节点在生成过程中与障碍区域发生冲突，则丢弃该节点并重新选择。当目标点包含在随机树的叶子节点中时，随机树停止扩展，可以获得从起始点到目标点的避障路线。详情参见快速探索随机树算法。
  • 人工势场法（ APF ）： 假设每架无人机都在人工势场中运动，目标点会对无人机产生引力场，而障碍物则会产生斥力场；在引力场和排斥场的共同作用下，无人机沿着势场方向生成可行轨迹。为了简化计算，首先计算引力势函数和斥力势函数的负梯度，并分别得到引力和斥力，进而得出两者的合力。基于合力计算姿态调整所需的控制量，并引导无人机避开障碍物，生成路径。人工势场法可与连续或离散环境建模技术结合使用。详情参见人工势场法。
  • Voronoi 图法：Voronoi 图是一种空间分割算法。该算法基于一系列种子节点（ Seed Points ）将空间划分为许多子区域，每个子区域被称为一个 Cell，Cell 中的所有点与该 Cell 的 Seed Points 之间的距离，小于到所有其他 Seed Points 的距离。 Voronoi 图根据障碍物的分布情况，对障碍物边之间的自由空间进行均分，在障碍物周围形成一个多边形，使得该多边形的边与障碍物之间的距离相等。显然，所有多边形的边一起构成了一个图模型，可以在源节点和目的节点之间通过图搜索得到满足需求的路径。
  • 概率路图算法（ Probabilistic Roadmap Algorithm ）：路径规划分为两个阶段，其中：在离线学习阶段随机采样大量飞行器的位姿点；然后，搜索邻居节点并建立连接以构建地标地图。在在线查询阶段，根据起点、目标点和地标地图信息，采用启发式搜索算法从地标地图中搜索可行轨迹。详情参见概率路图算法。
  • 可见性图算法（Visibility Graph）： 一种用于二维空间中的路径规划方法。通常也分为两个阶段，在地图构建阶段，将尝试生成一个图模型，其中节点代表空间中的特定位置（如障碍物的顶点和起点、终点），边表示两个节点之间不存在障碍物遮挡的直线路径；在路径搜索阶段，使用图搜索算法（如 A* 或 Dijkstra 等）找到从起点到终点的最短路径。详情参见可见性图算法。

应用场景

  • 常用于飞行器需要自主规划路径的场景，虽然几乎不可能获得最优解，但速度比较快。
  • 案例： 家用扫地飞行器、消费级的无人机

3.2 基于搜索的方法

方法特点

  • 初始条件
    • 起点和终点 + 环境（ 离散空间表示，即图结构 ）
  • 寻路过程
    • 通过相邻节点的展开和回溯找到路径
    • 可以设置一定的启发函数
  • 输出结果
    • 最优解、次优解或无解

典型算法

  • Dijkstra 算法： 一种经典的图算法，用于在加权图中计算从单一源点到所有其他节点的最短路径。该算法从起点开始进行搜索，每一步找出一个节点的最短轨迹，直至所有节点均遍历完成。详情参见Dijkstra 算法。
  • 深度优先搜索算法：一种用于图或树遍历的算法。它从图的某一个节点开始，尽可能深地搜索图的分支。当节点的所有邻接节点都已被访问过时，算法回溯到上一个节点，继续探索未被访问的节点。深度优先搜索通常使用递归或栈来实现。深度优先搜索算法能够高效地给出可行解，但不保证解的最优性。详情参见深度优先搜索算法。
  • 广度优先搜索算法：广度优先搜索算法能够找到最优解，但其计算效率较差。详情参见广度优先搜索算法。
  • 深度优先迭代深化搜索算法：该算法结合了深度优先搜索 (DFS) 的空间效率和广度优先搜索 (BFS) 的最优性。它执行一系列 “深度受限” 的搜索，从深度为 0 的约束开始，然后放宽限制，直到找到目标为止。这种方法确保搜索的完备性且最优性（就边数而言），同时还使用最少的内存。详情参见深度优先迭代深化搜索算法。
  • 经典 A* 搜索算法：一种广泛应用于路径规划和图搜索问题的启发式搜索算法。它结合了 最佳优先搜索（Best-First Search）和 Dijkstra 算法 的优点，通过使用启发式函数来引导搜索过程，以尽可能少的计算量找到从起点到目标的最短路径。详情参见经典 A* 算法。
  • 焦点搜索算法：焦点搜索是 A* 家族的一员，但考虑到其基础性，将其在此处单独列出。详情参见焦点搜索算法。
  • A* 家族： A* 算法存在无法任意角度、可扩展性差、动态适应性不足等不足。为了解决这些问题，人们开发了大量基于 A* 的新算法。我们将其整理在一起，并称之为 “A* 家族” ，详情参见经典 A* 算法。

面向个体的启发式/元启发式方法

特点

  • 初始条件
    • 起点和终点 + 环境 （ 离散空间表示，即图结构 ）
    • 需要一个初始的可行解
  • 寻路过程
    • 通过相邻节点的展开和回溯找到路径
  • 输出结果
    • 最优解、次优解或无解

基于物理的方法

  • 模拟退火算法 （ Simulated Annealing, SA ）：一种基于物理退火过程的随机优化方法，能够有效地在复杂的搜索空间中寻找全局最优解。它最初应用于组合优化问题，后来被广泛用于路径规划、调度问题、函数优化等多个领域。在路径规划中，模拟退火算法通过模拟金属物理退火过程的冷却过程，逐渐降低系统的 “温度” ，在高温时接受较差解以跳出局部最优解，而在低温时趋向于优化当前解。通过此退火机制，算法能够在复杂环境中找到一条近似最优的路径。详情参见模拟退火算法。
  • 引力搜索算法 （ Gravitational Search Algorithm ）： 一种启发式算法，遵循物理引力定律和运动定律。在所提出的算法中，智能体被视为对象，其性能通过其质量来衡量。所有这些物体都通过引力相互吸引，这种力导致所有物体向质量较重的物体整体移动。因此，每个质量都将通过重力获知并通过信息交换进行更新。详情参见引力搜索算法。
  • 混沌优化算法 （ Chaotic Optimization Algorithm ） ：一种全局优化方法，因其易于实现、执行时间短和摆脱局部最优的鲁棒机制等特点而引起人们的关注。 详情参见混沌优化算法。
  • 智能水滴算法 （ Intelligent Water Drops algorithm ） ： 一种基于群体的自然启发优化算法，其灵感来自于自然河流以及它们如何找到到达目的地的几乎最佳路径。一条天然河流在从源头到目的地的途中常常会在许多可能的路径中找到好的路径。详情参见智能水滴算法。
  • 磁优化算法 （ Magnetic Optimization Algorithm ）： 基于磁性粒子之间的磁性吸引与排斥原理，与 GSA 不同，磁性粒子基于不相关极性对给定粒子表现出吸引力和排斥力。这种双重功能对于平衡开发与探索的优化很有用。粒子被放置在晶格状结构中，被格子状结构群体中的相邻粒子吸引/排斥。这种概念的基本版本仅考虑长程吸引力，不考虑排斥力，已针对 14 个数字基准问题进行了测试。详情参见磁优化算法。

基于地理的方法

  • 禁忌搜索算法 （ Tabu Search Algorithm ）
    • 一种采用局部搜索的元启发式搜索方法，旨在通过一系列移动来探索所有可行解的搜索空间。每一次的移动都朝着可用方向中最好的方向进行。为了帮助算法摆脱局部（ 但不是全局 ）最小值，在每次迭代时禁止一组移动。形式上，该算法的主要元素是解表示、成本函数、邻域、禁忌列表、愿望准则和终止准则。详情参见禁忌搜索算法。
  • 帝国主义竞争算法 （ Imperialistic Competition Algorithm ）： 该算法建立在帝国主义竞争的基础上，以形成一个所有国家都具有相同地位的帝国，最后一个国家是优化问题的最佳结果和输出。详情参见帝国主义竞争算法。

面向群体的启发式/元启发式方法

特点

  • 初始条件
    • 起点和终点 + 环境 （ 离散空间表示，即图结构 ）
    • 需要一组初始的可行解
  • 寻路过程
    • 带有一定的启发函数
  • 输出结果
    • 近似最优解

进化类型算法

进化算法（ EA ）是在自然界生物进化的基础上总结出来的一种随机优化搜索算法。最广泛使用的算法是遗传算法（ GA ）。其主要思想是首先对飞行空间进行栅格化，找到被障碍物覆盖或与障碍物冲突的区域，然后在地图中随机生成起点。为了保证到达目标点的无碰撞轨迹，在轨迹规划过程中，将每条从起点到目标点的无碰撞轨迹表示为一个个体，每个个体都有一条染色体，因此每条无碰撞轨迹也可以成为染色体。轨迹中的每个片段都表示为一个基因。所有个体的集合，即从起点到目标点的所有生成的无碰撞轨迹，称为种群。我们设计相应的适应度函数来从种群中筛选出所需的个体。适应度高的个体是精英个体；通过精英个体之间的交叉变异操作，不断筛选出更优秀的精英个体，直至满足终止条件，最后留下的就是所需的避障路线。

  • 遗传算法 （ Genetic Algorithm ）
    • 一种基于自然选择机制的随机全局自适应搜索优化技术。它被初始化为包含许多染色体的种群，其中每个染色体代表问题的一个解，其性能由拟合度函数评估。遗传算法由三个主要阶段组成：选择、交叉和变异。这三个基本运算的应用可以创建比其父母更好的新个体。该算法重复很多代，最终停在问题的最优解处。
  • 进化规划算法 （ Evolutionary Programming ）
    • 从随机生成的候选解种群开始，经过多次代或迭代演变成更好的解，主要步骤是初始化、变异、竞争和选择。
  • 和声搜索算法 （ Harmony Search Algorithm ）
    • Harmony Search ( HS ) 是一种基于音乐的启发式优化算法。和声搜索算法模仿音乐即兴创作的过程，音乐家不断调整乐器的音高以实现更好的和声。全局轨迹规划问题的搜索过程与音乐即兴创作过程类似；即每个决策变量在搜索过程中不断更新自己的值，从而收敛到全局最优，获得最优轨迹
  • Differential Evolution algorithm ( DE )
  • Evolutionary Strategy ( ES )

群体智能类型算法

  • 粒子群优化算法 （ Particle Swarm Optimization ）
    • 粒子群优化（ PSO ）是一种通过模拟鸟类觅食行为而发展起来的具有种群搜索策略的自适应随机优化算法，用于解决工程和科学中的各种问题。粒子群优化将轨迹规划问题初始化为一组随机粒子，然后迭代寻找最优解。在每次迭代中，粒子通过跟踪个体和全局极值来更新其位置和速度，然后利用搜索空间完成最优轨迹规划。
  • 蚁群优化算法 （ Ant Colony Optimization Algorithm ）
    • 蚁群优化（ ACO ）是一种启发式全局优化算法，源自蚂蚁在寻找食物过程中的轨迹行为。蚁群算法利用蚂蚁的运动轨迹来表示待优化问题的可行解。整个蚁群的所有轨迹构成了待优化问题的解空间，轨迹较短的蚂蚁释放的信息素较多。较短轨迹上积累的信息素浓度逐渐增加，选择该轨迹的蚂蚁数量增加；最终所有蚂蚁都会在正反馈的作用下集中到最佳轨迹上，对应的轨迹就是问题的最优解
  • 人工蜂群法 （ Artificial Bee Colony ）
    • 人工蜂群算法（ ABC ）是一种模仿蜜蜂智能觅食行为的优化方法。算法流程如下：首先，将雇佣蜂分配给初始蜜源，并按照一定规则进行搜索，生成新的蜜源。然后，利用贪心选择方法保留适应度高的蜜源，并计算雇佣蜂找到的蜜源被跟随的概率。最后，用与雇佣蜂相同的方法跟随峰顶。如果蜜源满足被废弃的条件，则相应的雇佣蜂就成为侦察蜂，在搜索空间中随机搜索，生成新的蜜源，通过各个人工蜂个体的局部优化行为获得全局最优轨迹。
  • 细菌觅食优化算法 （ Bacteria Foraging Optimization algorithm ）
  • 布谷鸟搜索算法 （ Cuckoo Search algorithm ）
  • 珊瑚礁优化算法 （ Coral Reef Optimization algorithm ）
  • 基于教与学的优化算法 （ Teaching–Learning Based Optimization algorithm, TLBO ）
  • 萤火虫算法 （ Firefly Algorithm ）
  • 果蝇优化算法 （ Fruit Fly Optimization Algorithm ）
    • 果蝇优化算法（ FOA ）是一种基于果蝇觅食行为推导全局优化的新方法，利用果蝇在感官知觉方面优于其他物种的特点，特别是在嗅觉和视觉方面。首先，果蝇利用嗅觉来收集空气中的气味。然后飞到食物位置附近，利用视觉找到食物和同伴聚集的位置，并朝该方向飞行，从而实现解空间的群体迭代搜索，完成多无人机轨迹规划。
  • 混排蛙跳算法 （ Shuffled Frog Leaping Algorithm, SFLA ）
  • 鸽子启发式优化 （ Pigeon Inspired Optimization, PIO ）
    • 鸽子启发优化算法（ PIO ）是一种群体智能优化算法，旨在模拟鸽子的归巢行为。首先提出了三种鸽群优化模型：基于地磁场的地图模型、基于太阳的指针算子模型和基于地标算子模型的地标算子模型。其次，通过地图和指针算子识别出大致方向，然后利用地标算子修正当前方向，直至找到最佳轨迹。
  • 樽海鞘群算法 （ Salp Swarm Algorithm ）
    • 樽海鞘群算法（ SSA ）是一种基于樽海鞘在海洋中航行和觅食时的集群行为来推导和寻求全局优化的新方法。 SSA 算法通过在随机位置初始化多个樽海鞘来充当近似全局最优值。然后，计算每个樽海鞘的适应度，找到适应度最好的樽海鞘，将最好的樽海鞘的位置分配给一个变量，作为樽海鞘链追逐的源食物，并使用公式更新适应度系数。对于每个维度，迭代更新前导跳跃体和跟随跳跃体的位置，搜索空间确定全局最优轨迹。
  • 灰狼优化算法 （ Gray Wolf Optimization Algorithm ）
    • 灰狼优化算法（ GWO ）是受灰狼捕食行为启发而提出的一种新型群体智能优化算法。灰狼优化算法根据计算出的适应度将灰狼个体适应度分为四个不同级别：最优解、次优解、第三解和候选解；适应度排名前三的个体引导其他狼向目标点迭代搜索，同时不断更新解的层次和位置，直到找到到达目标点的最佳轨迹
  • 模因算法
