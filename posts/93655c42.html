密集建筑环境中无人机配送的路径规划

Created2025-01-09|Updated2025-01-17|空域管理空域结构
|Word Count:6.6k|Reading Time:22mins

【摘 要】无人机已被引入城市环境以方便我们的生活，例如货物运送服务。但城市地区密集的建筑物对无人机的安全运行构成挑战，因为与建筑物相撞的风险。为了应对这一挑战，我们提出了一种利用改进的蚁群优化 ( IACO ) 算法的路径规划方法。该算法通过自适应参数机制和信息素强度更新机制改进了 ACO 的标准设置。通过引入基于快速探索随机树 ( RRT ) 的机制来提高搜索效率，从而进行了进一步的改进。仿真结果表明，我们提出的方法显著提高了复杂城市环境中路径规划的收敛速度和解决方案的质量。它可以在二维和三维环境中始终如一地以更快的收敛速度产生令人满意的解决方案。

【参考资料】

引言

无人机已被应用于城市环境中的物流、巡检、疫情防控等各个领域 [5]。对于无人机物流应用，飞行路径的规划至关重要，安全有效的航线是成功完成配送任务的前提 [13]。现有的研究工作从多个角度开展了无人机路径规划问题。以最小化飞行距离为目标，提出了一种选择性目标快速探索随机树算法来生成更短路径 [9]。提出了一种考虑飞行时间的固定翼无人机在线路径规划算法 [2]。但以前的大多数研究都忽视了无人机运行环境中风险成本对路径规划的影响，这可能导致规划路径穿越高风险区域 [6, 13]。

其他研究工作则研究了二维或三维环境中的路径规划算法。例如，研究人员提出了一种在室外三维环境中为无人机进行路径规划和避障的方法 [7]，目的是减少飞行过程中的能量消耗。另一项研究将路径规划问题建模为静态二维空间中的优化问题，同时采用自然启发算法来避开障碍物 [4]。但它的适用性仅限于这些环境，因此其通用性受到限制。

现有研究主要集中在特定二维或三维属性的连续环境中的路径距离优化，适用于风险环境并考虑路径风险成本的无人机路径规划方法较少。针对这一问题，本文以考虑风险的飞行路径成本优化为目标，提出了一种基于 RRT 与蚁群算法融合的增强方法。通过定义无人机运行环境和飞行规则，建立路径规划模型。改进蚁群算法，设计 RRT 算子，并提出将 RRT 算子与增强蚁群算法融合，使其能在二维和三维环境中以相对较快的计算效率生成低风险路径。

路径规划模型

虽然飞行器自由飞行的概念已经被提出，但是出于安全考虑，航空领域要求飞行器遵循由航路点组成的预定路径。此外，为了更好地管理空中交通，本文将以离散化的方式处理路径规划问题。我们采用空间分解方法将无人机的运行空域划分为规则形状的单元，具体为大小相等的立方体 [13]。无人机需要朝着立方体单元的顶点飞行，飞行的航段将沿着 “立方体的边” 、 “体对角线” 或 “面对角线” 飞行，如 所示。

在这种环境下，任何包含障碍物的单元都被视为已占用。无人机需要避开这些充满障碍物的区域，只在无障碍区域飞行。为了解决描述障碍物不规则形状的问题，更好地使无人机避开障碍物从而提高运行安全性，单元中未被障碍物完全占用的空间也被视为障碍物空间。基于该运行环境，本文的决策变量是无人机飞过的航路点。约束和拟合度函数如下面两个小节所述。

约束

飞行运动学约束：无人机只能在指定的空域内飞行。

\[\left \{\begin{array}{l} x_{i+1}=\left \{x_{\min}+l \cdot i \mid i=0,1, \ldots, \frac{x_{\max}-x_{\min}}{l}\right \}\tag{1} \\ y_{j+1}=\left \{y_{\min}+l \cdot j \mid j=0,1, \ldots, \frac{y_{\max}-y_{\min}}{l}\right \} \\ z_{k+1}=\left \{z_{\min}+l \cdot k \mid k=0,1, \ldots, \frac{z_{\max}-z_{\min}}{l}\right \} \end{array}\right.\]

其中 \(\left ( x_{i+1}, y_{i+1}, z_{i+1}\right )\) 表示飞行路径经过的航路点坐标。\(\left ( x_{\min}, x_{\max}\right )\)、\(\left ( y_{\min}, y_{\max}\right )\) 和 \(\left ( z_{\min}, z_{\max}\right )\) 表示可飞行空域的边界， \(l\) 表示立方体单位的边长。

避障约束：飞行过程中，无人机需绕行障碍物，避免发生碰撞。

\[\sqrt{\left ( x_{n}-x_{o}\right ) ^{2}+\left ( y_{n}-y_{o}\right ) ^{2}+\left ( z_{n}-z_{o}\right ) ^{2}}\neq 0 \tag{2}\]

其中 \(\left ( x_{n}, y_{n}, z_{n}\right )\) 表示航路点的坐标， \(\left ( x_{o}, y_{o}, z_{o}\right )\) 表示障碍物空间内任意点的坐标。

飞行时间约束：由于无人机的续航能力有限，因此需要对其飞行时间进行约束。此约束可以表示为基于网格的环境中无人机飞行路径中包含的航路点数量的限制。

\[N \leq N_{\max}\tag{3}\]

其中 \(N\) 表示路径中的航路点总数，而 \(N_{\max}\) 表示路径允许的最大航路点数量。

拟合度函数

无人机可能失控坠落，对地面人员和财产造成风险 [6, 13]。为了降低运行风险，拟合度函数被设计为与风险成本相关的表达式。

\[R=\sum_{ ( i, j, k ) \in \Theta}R_{i, j, k}\tag{4}\]

其中， \(R\) 表示路径的总风险成本，即无人机飞过的每个单元的风险成本值之和。总风险成本较低的路径被认为是更好的解决方案；注意， \(\Theta\) 是无人机经过的单元集合； \(( i, j, k )\) 代表每个单元在运行环境模型中的位置索引（ 如 所示 ）， \(R_{i, j, k}\) 对应于单元的风险值。

离散路径优化算法的实现

智能优化算法模拟生物行为，由于其强大的优化能力和易于与特定研究问题相结合，近年来已成为强大的优化工具。因此，它们已被用于解决无人机路径规划问题 [15, 16]。但大多数智能算法都是为解决连续空间内的优化问题而设计的，很少有适合在离散环境中使用的 [10, 16, 17]。另一方面，蚁群优化（ ACO ）算法是离散环境中常用的算法 [1, 14]。但基本的 ACO 在收敛速度和局部最优方面存在问题。为了解决这些问题，我们提出了一种改进的蚁群优化算法（ IACO ），以提高算法的收敛速度和全局优化能力。此外，为了提高算法的探索能力，针对离散环境的特点，设计了一个快速探索随机树（ RRT ）算子，并将 RRT 算子与 IACO 算法相结合，得到一种 RRT 算子与改进蚁群优化算法相结合的混合算法（ 即一种基于 RRT 的 IACO 算法，简称 RIACO ）。

改进的蚁群优化算法（ IACO ）

本节从路点选取、参数自适应机制、信息素强度更新机制三个方面对标准 ACO 进行了改进，具体改进如下。

航路点选择：从当前航路点过渡到下一个航路点时，选择航路点的概率为

\[P ( t ) =\frac{\left[\tau_{n} ( t ) \right]^{\alpha}\left ( \eta_{n}\right ) ^{\beta}}{\sum_{\varepsilon \in \text{all}}\left[\tau_{\varepsilon} ( t ) \right]^{\alpha}\left ( \eta_{\varepsilon}\right ) ^{\beta}}, t=1,2, \ldots, T \tag{5}\]

其中， \(P ( t )\) 为第 \(t\) 次迭代中某个路径点 \(n\) 被选中的概率， \(\tau\) 表示信息素强度， \(\alpha\) 表示信息素因子， \(\eta\) 表示启发式值， \(\beta\) 表示启发式因子，all 为所有满足约束条件的潜在路径点集合， \(T\) 为最大迭代次数。原始蚁群算法中的启发式值为当前路径点到端点距离的倒数。但从安全角度考虑，得到的路径总风险成本应该更低。因此，本文将启发式项修改为

\[\eta_{n}^{\text{des}}=\left \{\begin{array}{l} 1 / \sqrt{\left ( R_{n, x}^{\text{des}}\right ) ^{2}+\left ( R_{n, y}^{\text{des}}\right ) ^{2}+\left ( R_{n, z}^{\text{des}}\right ) ^{2}}, n \neq \operatorname{des}\tag{6} \\ 1, n=\text{des} \end{array}\right.\]

其中 \(R_{n, x}^{\text{des}}, R_{n, y}^{\text{des}}\)、和 \(R_{n, z}^{\text{des}}\) 分别表示从候选航路点到 \(x, y\)、和 \(z\) 轴处的路径端点 des 的风险估计值。

另外，在 ACO 算法中，航路点的选择采用标准轮盘赌选择过程来确定，由于标准轮盘赌选择的随机性较大，使得算法具有一定的全局搜索能力，但这也降低了算法的收敛速度。为了解决这个问题，提高算法的收敛速度，我们对 IACO 算法中的标准轮盘赌选择进行了改进，在航路点选择过程中每次迭代都预先生成一个随机数，当该随机数超过某个阈值时，采用标准轮盘赌方法来确定航路点的选择，否则直接选择启发式值最低的候选航路点作为无人机位移的下一个航路点。

通过引入随机数，直接选取启发值最小的航路点并以一定的概率加入到路径中，为蚂蚁的路径搜索过程提供了方向性，提高了搜索效率和收敛速度。同时，为避免算法陷入局部最优，IACO 的整个迭代过程保持了算法搜索的随机性。我们不仅在 IACO 迭代初期采用改进的轮盘赌选择法来选择航路点，而且在整个迭代过程中始终应用该改进方法。这一改进使算法避免陷入局部最优，加快了算法的收敛过程。

参数自适应：ACO 算法中的信息素强度是常数，在整个迭代过程中保持不变。如果信息素强度设置得太低，会增强算法的全局搜索能力，但会降低收敛速度；反之，如果信息素强度设置得太高，算法可能陷入局部最优。为了使算法能够以更快的收敛速度搜索到更高质量的解，算法需要在迭代开始时提高全局搜索能力，并在迭代过程中解决局部搜索问题。因此，本文提出的改进 IACO 算法对信息素强度进行了自适应调整，在迭代开始时设置一个较低的强度值，随着迭代的进行逐渐增大。

\[Q ( t ) =Q_{0}+\ln ( t ) g \tag{7}\]

其中 \(Q\) 为信息素强度， \(Q_{0}\) 为初始强度值， \(g\) 为增强系数。

信息素强度更新：不同代蚂蚁通过信息素传递信息，每代蚂蚁根据上一代蚂蚁留下的信息素选择路径点。因此，在每一代路径生成后，ACO 算法需要更新每个路径点的信息素强度，记为

\[\left \{\begin{array}{l} \tau_{n} ( t+1 ) = ( 1-\rho ) \tau_{n} ( t ) +\Delta \tau_{a}, n \in \Lambda_{a}\tag{8} \\ \Delta \tau_{a}=Q ( t ) / R_{a}, a=1,2, \ldots, A \end{array}\right.\]

其中， \(\rho\) 为信息素蒸发系数， \(\Lambda_{a}\) 为蚂蚁经过的路点集合， \(a, R_{a}\) 为该蚂蚁所走路径的拟合度值， \(\Delta \tau_{a}\) 为信息素强度的增加值， \(A\) 表示种群大小。

原有的 ACO 算法会更新所有路径航路点的信息素强度，这导致后续蚂蚁在搜索路径时会受到先前较差解的影响，对算法的收敛没有贡献。为了解决这个问题，提高收敛速度，我们只更新当前代最佳路径中航路点的信息素强度。这种改进的好处是蚂蚁的搜索过程只受到前几代优质解的影响，而不会受到较差解的干扰，从而提高算法的搜索效率。但这种做法也增加了算法重复搜索先前出现过的解的可能性，这很容易导致算法陷入局部最优。为了避免这种情况，对信息素设置了一个最大阈值约束，记为

\[\tau \leq \tau_{\max}\tag{9}\]

其中 \(\tau_{\max}\) 是信息素的最大强度。

离散 RRT 算法设计

RRT 算法是一种具有概率完备性的路径搜索算法，可以快速搜索一条从起点到终点满足所有约束条件的无人机飞行路径。为了使蚁群优化算法的解质量更高，提高运行效率，设计了一个 RRT 算子与蚁群算法相结合。但由于 RRT 算法的标准形式是在连续环境下工作的，因此本文对其进行了改进，使其适应于离散环境。

RRT 算法的标准形式遍历所有搜索空间 [9]。为了便于算法收敛，修改后的 RRT 算法以一定的概率选取路径的终点作为扩展的随机点。随机点的生成也必须符合 所示的离散环境模型，即只能在每个立方单元的顶点生成。在扩展生成树的过程中，如果存在终点为候选航路点的情况，则直接选取该终点加入生成树；否则，选取随机点与候选航路点中风险成本最低的航路点加入生成树，完成一次生成树扩展过程。重复上述扩展过程，直至路径终点加入生成树，停止扩展，生成一条从起点到终点的可行飞行路径。

融合 RRT 算子的改进蚁群优化算法（ RIACO ）

本文将 RRT 算法与 IACO 算法进行融合，为蚁群算法的迭代过程提供更好的进化方向，加快算法的收敛速度，增强算法的全局寻优能力，集成算法命名为 RIACO，算法的工作流程如 所示。

RIACO 算法增加了初始化过程，与上述增强型 IACO 算法相比，进一步改进了路径点选择策略，同时提高了初始解的质量，从而提高了收敛速度。ACO 算法和 IACO 算法中，蚂蚁在迭代开始时是随机游走且无引导，导致初始路径曲折且拟合度函数值较高。由于历史路径的质量影响后续进化过程中解的质量，因此 RIACO 算法增加了初始化过程，为进化提供更好的方向。利用改进的 RRT 算法，从起点到终点初始化多条可行路径，并更新信息素强度矩阵，为蚁群后续迭代过程提供粗略的进化方向。该方法有助于提高每代解的质量，加快收敛速度。

蚁群优化算法中解的搜索过程受到历史解和终点对应的启发项的影响，导致蚁群优化算法容易陷入局部最优。为了解决该问题，进一步增强算法的全局搜索能力，本文将 RRT 算子融入航路点选择机制中，提出一种对应于随机点的航路点选择策略，以拓宽解的搜索范围。

随机点选取方法在 3.2 节中描述，其中插入一个随机点，并将随机点对应的 RIACO 算法的启发项表示为

\[\eta_{n}^{r a n}=\left \{\begin{array}{l} 1 / \sqrt{\left ( R_{n, x}^{r a n}\right ) ^{2}+\left ( R_{n, y}^{r a n}\right ) ^{2}+\left ( R_{n, z}^{r a n}\right ) ^{2}}, n \neq r a n \tag{10} \\ 1, n=r a n \end{array}\right.\]

其中， \(R_{n, x}^{r a n}, R_{n, y}^{r a n}\)、\(R_{n, z}^{r a n}\) 分别为候选航路点到随机点三个坐标方向上的成本估算值。将公式（ 10 ）与公式（ 5 ）结合，即可得到与随机点对应的航路点选择方法。

RIACO 算法借鉴了第三节 IACO 的路径点选择机制，采用了一定概率随机点对应的路径点选择方法。与传统的只以端点为目标点的路径搜索方法相比，本文引入了 RRT 算法的随机性，增加了解的搜索范围。这一改进提高了找到高质量解的机会，避免了算法陷入局部最优。

模拟研究

为了证明所提路径规划方法的有效性，我们将其与原始 ACO、改进版 IACO 和集成 RRT 算子的 RIACO 进行了比较。比较研究的结果证明了本文提出的改进策略的有效性。此外，在二维和三维密集建筑环境中的案例研究证明了改进算法的通用性。

二维环境案例分析

本节采用 ACO、IACO 和 RIACO 算法解决二维密集建筑物环境中的路径规划问题。二维模拟环境的范围为 \(\mathrm{x}=[0,1800] \mathrm{m}\) 和 \(y=[0,1800] \mathrm{m}\) ，每个单元格的风险成本指数为 \([1,50]\) 内的随机数。模拟环境中随机生成 300 个建筑物障碍物。每个立方体的边长为 30m，单条飞行路径的最大航路点数限制为 500 个。蚁群优化算法的参数如 所示。

蚁群优化算法的参数
Parameter Value Parameter Value
\(\alpha\) 0.5 \(\beta\) 1
\(T\) 200 \(A\) 100

从 可以看出，ACO 算法在前 200 代没有表现出明显的收敛趋势，而 IACO 和 RIACO 算法的收敛速度明显提升，分别在第 61 代和 43 代收敛。除了收敛性能上的优势外，改进后解的拟合度函数值也比改进前有所降低，IACO 和 RIACO 算法在收敛点处的拟合度函数值分别为 1462.5 和 1435。

改进算法之所以能在收敛速度和解质量上体现出明显的优势，是因为相比于标准轮盘选择方法，航路点选择过程得到了增强，为算法的搜索过程提供了指导。

此外，从计算开始，自适应算法参数机制赋予算法强大的全局搜索能力，有利于早期探索和发现高质量解；而在后期，通过增强算法的局部搜索能力来提高算法的收敛性。此外，我们根据蚂蚁的进化过程和代际通信方式增强了信息素强度的更新机制。这消除了低质量解对路径搜索的不利影响。同时，限制信息素强度的范围也可以防止搜索过程陷入局部最优。此外，与 IACO 算法相比，RIACO 算法表现出更高的初始拟合度函数值，但在后期产生了更高质量的解。这种改进归功于 RIACO 算法的初始化过程和以随机点为基准的路径点选择机制。这扩大了解的搜索范围，从而获得了更高的初始拟合度函数值。但它的优点是增强了算法的全局搜索能力，增加了算法找到高质量解的可能性。

三维密集建筑环境中的模拟实验

另一组仿真在三维环境中进行，以证明所提方法的泛化能力。模拟边界为 \(x=[0,1800] m\)、 \(y=[0,1800] m\)、\(z=[0,90] \mathrm{m}\) ，每个单元的风险成本指数为从 \([1, 50]\) 中选取的随机数。共随机产生 300 个三维建筑障碍物。无人机配送任务的起点为 \(( 1680,240,60 )\) ，终点为 \(( 150,1590,90 )\)。信息素强度初值取为从起点到终点三个方向的成本值之和。其他参数与二维环境中设置的相同。ACO、IACO、RIACO 算法的收敛曲线如 所示，IACO 和 RIACO 算法找到的路径如 所示。

如 所示，ACO 算法在三维环境下的收敛速度仍然较慢，但经过改进，我们提出的 IACO 和 RIACO 算法分别在第 55 代和第 50 代实现收敛，对应的拟合度函数值分别为 1428 和 1396。

在三维环境下，改进前后算法的性能对比与二维环境下的性能一致，改进算法 IACO 和 RIACO 在收敛速度和解质量上均较 ACO 算法有明显优势，其中融合 RRT 算子的 RIACO 算法性能最优。提出的无人机路径规划模型与算法在二维和三维密集建筑障碍环境下均适用，能够以更快的收敛速度得到更高质量的解。

结论

本文重点研究无人机配送任务在复杂城市环境中的路径规划问题，其中密集的建筑物对无人机安全导航构成了挑战。为了应对这一挑战，提出了一个使用离散表达方法的数学模型，目标是最小化与路径规划问题相关的总风险成本。为了解决该模型，首先将原始蚁群优化 ( ACO ) 算法增强为改进的蚁群优化 ( IACO ) 算法。这是通过结合航路点选择机制、自适应参数机制和信息素更新机制来实现的。这些增强功能提高了解决路径规划问题的收敛速度。此外，为了进一步提高解决方案的质量，开发了一种 RIACO 算法，并在 IACO 算法中设计并引入了改进的 RRT 算子。这些创新将随机性引入路径搜索过程，增强了算法的全局搜索能力。此外，还包括一个初始化过程来指导算法的迭代方向。模拟结果证明了所提出方法在二维和三维密集建筑环境中的有效性和通用性。对比分析表明，改进的 IACO 算法与 RIACO 算法在收敛速度、解质量等方面较原 ACO 算法有明显的优势。

这项工作可以向多个方向扩展。一个潜在的研究课题是基于风险的动态路径优化，考虑第三方对地面人员的风险 [3, 11]。这一扩展将有助于降低城市环境中无人机运行风险下第三方人员的安全风险。后续工作还可以探索具有冲突检测和解决的四维轨迹优化问题，以生成无冲突的多无人机飞行轨迹 [8, 12]。
